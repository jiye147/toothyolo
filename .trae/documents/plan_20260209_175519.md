# 牙齿疾病智能检测系统 - 公网部署计划

## 📋 部署方案选择

### 方案1: 免费云平台部署（推荐新手）
**平台**: Vercel / Netlify / Render
**优点**: 完全免费、自动HTTPS、全球CDN
**缺点**: 后端API需要单独部署

### 方案2: 付费云服务器部署（推荐生产环境）
**平台**: 阿里云 / 腾讯云 / AWS / DigitalOcean
**优点**: 完全控制、性能好、可扩展
**缺点**: 需要付费、需要配置

### 方案3: Docker容器化部署
**平台**: 任何支持Docker的云平台
**优点**: 环境一致、易于迁移
**缺点**: 需要Docker知识

## 🎯 推荐方案：免费部署（Vercel + Render）

### 前端部署（Vercel）
1. 注册Vercel账号
2. 连接GitHub仓库
3. 自动部署
4. 获得免费域名

### 后端部署（Render）
1. 注册Render账号
2. 连接GitHub仓库
3. 配置环境变量
4. 自动部署
5. 获得API地址
